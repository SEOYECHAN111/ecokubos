개인정보 처리방침

자원잇다는 개인정보 보호를 서비스 운영의 핵심 원칙으로 두며, 이용자의 개인정보를 필요한 범위에서만 처리합니다. 본 방침은 총칙, 처리 목적, 처리 항목, 보유기간, 제3자 제공, 처리위탁, AI·OCR·지도·결제 API 이용, 정보주체 권리, 파기, 안전성 확보조치, 아동 개인정보, 위치정보, 국외이전 가능성, 문의처 및 개정 절차를 포함합니다.

1. 총칙

본 서비스는 개인 수거신청, 기관·기업 서류 자동작성, 업체 작업입찰, 관리자 운영관리를 위해 최소한의 정보를 처리합니다. 온비드 ID, 비밀번호, 공동인증서 파일, 인증서 비밀번호는 수집하지 않습니다.

2. 처리하는 개인정보 항목

  • 개인회원: 성명, 연락처, 수거지역, 품목사진, 수거요청 내용
  • 업체회원: 사업자 정보, 담당자 정보, 허가·신고자료, 차량·보험자료
  • 기관·기업회원: 기관명, 담당자 정보, 공문·목록표·사진자료, 업무요청 내용
  • 관리정보: 접속기록, 다운로드 이력, 문의내용, 결제상태, 감사로그

3. AI·OCR·지도·결제 API

AI 자동작성과 OCR 분석 결과는 담당자 검토용 초안이며 자동 확정되지 않습니다. 지도·노선 API는 수거 위치와 동선 계산에 사용되며, 결제 API는 요금제 결제와 환불 처리에 사용됩니다. API 키는 프론트엔드에 노출하지 않고 서버 환경변수로 관리합니다.

4. 보유 및 파기

업무 목적이 달성되거나 보유기간이 경과하면 지체 없이 파기합니다. 다만 계약, 정산, 분쟁 대응, 법령상 보관 의무가 있는 경우 해당 기간 동안 분리 보관할 수 있습니다.

5. 안전성 확보조치

접근권한 분리, 보호자료 권한관리, 파일 확장자·용량 제한, 다운로드 로그, 개인정보 마스킹, 관리자 감사로그, 비밀키 서버 보관, 보안 헤더 적용 등을 운영합니다.

6. 정보주체 권리

이용자는 열람, 정정, 삭제, 처리정지, 동의철회를 요청할 수 있습니다. 다만 계약 이행, 정산, 분쟁 대응 등에 필요한 정보는 관련 기준에 따라 일정 기간 보관될 수 있습니다.

공식 공고·최종 입찰은 온비드에서, 전후 서류·수거·반출·성과관리는 자원잇다에서 준비합니다.
