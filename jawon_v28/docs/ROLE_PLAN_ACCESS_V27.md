# v27 역할·요금제 접근 설계

공개 홈페이지에서는 핵심 소개와 가입·문의만 노출하고, 실제 돈이 되는 업무 기능은 로그인 후 역할과 플랜에 따라 노출한다.

| 기능 | 공개 노출 | 개인 | 업체 | 기관 | 기업 | 관리자 |
|---|---|---:|---:|---:|---:|---:|
| 사진수거 | 가능 | Free 이상 | - | - | - | 전체 |
| 입찰방 | 불가 | - | Basic 이상 | - | - | 전체 |
| 수거지도 | 불가 | - | Standard 이상 | - | Standard 이상 | 전체 |
| 최적노선 | 불가 | - | Plus 이상 | - | ESG Plus 이상 | 전체 |
| AI 서류작성 | 불가 | - | - | Basic 이상 | Basic 이상 | 전체 |
| 보호자료센터 | 불가 | - | Standard 이상 | Basic 이상 | Standard 이상 | 전체 |
| 관리자 관제 | 불가 | - | - | - | - | 전체 |

상단 메뉴는 로그인 전과 로그인 후가 다르다. 로그인 전에는 `기관/업체/기업/입찰방/최적노선`을 직접 노출하지 않고, 로그인 후 사용자 권한에 맞는 업무 메뉴를 구성한다.
